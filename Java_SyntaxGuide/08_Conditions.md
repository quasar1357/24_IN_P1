# Conditions
[Back to overview](./00_Java_SyntaxGuide.md)

## Simple **if**
```java
if (arbBool)
    ... // single line!

// OR

if (arbBool) {
    ... // multiple lines!
    ... // multiple lines!
}
```

## If with **else**
```java
if (arbBool)
    ...
else
    ...

// OR

if (arbBool){
    ...
    ...
} else {
    ...
    ...
}
```

## If with **else if** and else
```java
if (arbBool)
    ...
else if (anotherBool)
    ...
else
    ...

// OR

if (arbBool){
    ...
    ...
} else if (anotherBool) {
    ...
    ...
} else {
    ...
    ...
}
```
